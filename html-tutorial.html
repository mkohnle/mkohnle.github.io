Statische Restaurant-Website Tutorial

Schritt 1: Projektordner erstellen

Erstelle einen neuen Ordner für das Projekt und benenne ihn nach deinem Restaurant. Zum Beispiel "CooleRestaurantWebsite".

Lade die Bilder hier herunter.

Schritt 2: Dateien erstellen

Erstelle drei neue Dateien im Projektordner:

  • index.html
  • css/style.css
  • images/

Schritt 3: HTML-Grundstruktur

Füge den folgenden Code in deine index.html-Datei ein, um die grundlegende HTML-Struktur zu erstellen:

<!DOCTYPE html>
<html>
<head>
<title>Dein Restaurantname</title>
</head>
<body>

<!-- Dein Inhalt hier einfügen -->

</body>
</html>

Schritt 4: Verknüpfe die CSS-Datei

Füge den folgenden Code im <head>-Bereich deiner index.html-Datei ein, um die CSS-Datei zu verknüpfen: 

<link rel="stylesheet" href="css/style.css">

Schritt 5: Erstelle die Navigationsleiste

Füge den folgenden Code innerhalb des <body>-Bereichs ein, um die Navigationsleiste zu erstellen:

<div class="navbar">
  <a href="#" class="active">HOME</a>
  <a href="#menu">MENU</a>
  <a href="#about">ABOUT</a>
  <a href="#contact">CONTACT</a>
</div>

Schritt 6: CSS-Styling

Füge den folgenden CSS-Code in die "style.css"-Datei ein, um die Navigationsleiste zu stylen:

/* Navigationsleisten-Styling */
.navbar {
  background-color: #333;
  overflow: hidden;
}

.navbar a {
  float: left;
  display: block;
  color: white;
  text-align: center;
  padding: 14px 16px;
  text-decoration: none;
}

.navbar a:hover {
  background-color: #ddd;
  color: black;
}

.navbar a.active {
  background-color: #4CAF50;
  color: white;
}

Schritt 7: Erstelle die Sections für Menü, About und Kontakt

Füge den folgenden Code in deine index.html-Datei ein, um die Sections zu erstellen:

<section id="menu">
  <h1>THE MENU</h1>
  <!-- Füge hier den Inhalt für das Menü ein -->
</section>

<section id="about">
  <h1>ABOUT</h1>
  <!-- Füge hier den Inhalt für den About-Bereich ein -->
</section>

<section id="contact">
  <h1>CONTACT</h1>
  <!-- Füge hier den Inhalt für den Kontakt-Bereich ein -->
</section>

Schritt 8: Erstelle den Inhalt für das Menü

Füge den folgenden Code innerhalb des <section id="menu"> ein, um den Menü-Inhalt zu erstellen:

<h2>Pizza</h2>
<p>Frische Tomaten, frische Mozzarella, frisches Basilikum</p>

<h2>Pasta</h2>
<p>Spezialsauce, Mozzarella, Parmesan, Rinderhackfleisch</p>

<h2>Starter</h2>
<p>Brot mit Pesto, Tomaten, Zwiebeln, Knoblauch</p>

Schritt 9: Erstelle den Inhalt für den About-Bereich

Füge den folgenden Code innerhalb des <section id="about"> ein, um den Inhalt für den About-Bereich zu erstellen:

<h2>About</h2>
<p>Das Restaurant wurde von Herrn Italiano in blabla gegründet. Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>
<p><strong>Der Chef?</strong> Herr Italiano persönlich<img src="images/chef.jpg" style="width:150px" alt="Chef"></p>
<p>Wir sind stolz auf unsere Inneneinrichtung.</p>

Schritt 10: Erstelle den Inhalt für den Kontakt-Bereich

Füge den folgenden Code innerhalb des <section id="contact"> ein, um den Inhalt für den Kontakt-Bereich zu erstellen:

<h2>Contact</h2>
<p>Finde uns an einer Adresse an einem Ort oder rufe uns unter 05050515-122330 an</p>
<p><span class="tag">FYI!</span> Wir bieten Catering für Veranstaltungen jeglicher Größe an. Wir verstehen deine Bedürfnisse und stellen das Essen zufrieden, sowohl optisch als auch geschmacklich.</p>
<p><strong>Reserviere</strong> einen Tisch, frage nach dem Tagesangebot oder sende uns einfach eine Nachricht:</p>
<form action="/action_page.php" target="_blank">
  <p><input class="input" type="text" placeholder="Name" required name="Name"></p>
  <p><input class="input" type="number" placeholder="How many people" required name="People"></p>
  <p><input class="input" type="datetime-local" placeholder="Date and time" required name="date" value="2020-11-16T20:00"></p>
  <p><input class="input" type="text" placeholder="Message \ Special requirements" required name="Message"></p>
  <p><button class="button" type="submit">SEND MESSAGE</button></p>
</form>

Schritt 11: Erstelle den Footer

Füge den folgenden Code am Ende deiner index.html-Datei ein, um den Footer zu erstellen:

<footer>
  <p>© Dein Restaurantname 2023</p>
</footer>

Schritt 12: CSS-Styling für Kontakt-Bereich

Füge den folgenden CSS-Code in die "style.css"-Datei ein, um das Styling für den Kontakt-Bereich festzulegen:

/* Kontaktformular-Styling */
.input {
  padding: 8px;
  width: 100%;
  margin-bottom: 16px;
}

.button {
  background-color: #333;
  color: white;
  padding: 12px 16px;
  border: none;
  cursor: pointer;
  width: 100%;
  font-size: 18px;
}

.tag {
  background-color: #333;
  color: white;
  padding: 4px 8px;
  font-size: 16px;
}

Schritt 13: Vorschau

Öffne deine "index.html"-Datei in einem Webbrowser, um die statische Restaurant-Website zu sehen. Überprüfe, ob die Navigationsleiste funktioniert und die Menüpunkte "MENU", "ABOUT" und "CONTACT" zu den entsprechenden Sections scrollen.
